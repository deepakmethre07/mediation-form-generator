# FORM A – Mediation Application Generator

This project generates an MS Word document for **FORM ‘A’ – Mediation Application Form**
using Python, accurately replicating the layout and structure of the provided PDF.

## Tech Stack
- Python 3.x
- python-docx

## Features
- Exact table-based layout
- Legal placeholders preserved
- Clean and readable Word output
- Matches official FORM A structure

## How to Run
```bash
pip install python-docx
python form_a_generator.py
